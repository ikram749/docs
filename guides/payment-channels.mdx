---
title: "Payment Channels"
---

SayabeliPay supports multiple payment channels.

| Channel          | Description                                      |
|------------------|--------------------------------------------------|
| `ONLINE_BANKING` | FPX online banking payment within Malaysia       |
| `TOUCH_N_GO`     | Payment via Touch 'n Go eWallet                  |
| `SHOPEE_PAY`     | Payment via ShopeePay eWallet                    |


👉 When using `ONLINE_BANKING`, the `payment_code` field is optional. If provided, it will preselect the customer's bank from the supported bank list.
