When the `channel `is set to **ONLINE_BANKING** or **WALLET**, you must provide a `payment_code`.

### FPX Bank Codes

To fetch the full list of banks and codes dynamically, call the [FPX Bank Listing API](https://sayabelipay.mintlify.app/api-reference/retrieve-fpx-bank):

### Supported Wallets

| Payment Code | Description       |
| ------------ | ----------------- |
| `ALIPAY`     | Alipay (CN)       |
| `WECHAT`     | WeChat Pay (CN)   |
| `BOOST`      | Boost Wallet      |
| `TNG`        | TNG eWallet       |
| `GRABPAY`    | GrabPay           |
| `MAE`        | MAE eWallet       |
| `SHOPEEPAY`  | ShopeePay eWallet |
